# 🐊 귀여운 악어 슈터

오프라인에서도 곧장 실행할 수 있는 HTML5 캔버스 기반 미니 게임입니다. 2000년대 플래시 감성을 살린 한국어 인터페이스와 귀여운 악어 루루가 사막 하늘을 가르는 새 떼를 막아냅니다.

## 🎮 온라인 플레이

GitHub Pages로 호스팅된 게임을 바로 플레이할 수 있습니다: [게임 플레이하기](https://[사용자명].github.io/cute-croc-shooter)

## 📁 구성 파일

- `index.html` : 게임 레이아웃과 UI, 한국어 설명을 포함한 메인 문서
- `styles.css` : 레트로 감성을 살린 글꼴과 HUD, 페이지 전체 스타일
- `game.js` : 캔버스 렌더링, 입력 처리, 총알/적/파티클 로직이 포함된 게임 엔진

## 🚀 실행 방법

### 로컬 실행
1. 이 폴더(`cute-croc-shooter`)를 그대로 유지한 뒤, `index.html`을 더블클릭해 브라우저에서 여세요.
2. 별도의 서버 없이 최신 브라우저에서 바로 플레이 가능합니다. (Chrome, Edge, Safari 권장)

### GitHub Pages로 배포
1. 이 저장소를 GitHub에 업로드
2. 저장소 Settings > Pages로 이동
3. Source를 `main` 브랜치의 `/ (root)`로 설정
4. 저장 후 몇 분 후 `https://[사용자명].github.io/cute-croc-shooter`에서 접속 가능

## 게임 룰

- 하늘에서 내려오는 새를 처치하면 기본 50점을 획득하고, 연속 처치 시 콤보 배수가 증가합니다. 다만 10라운드부터는 2회, 20라운드부터는 3회 명중해야 쓰러집니다.
- 체력은 3 ❤️이며 모두 소진하면 GAME OVER.
- 최고 점수는 브라우저 `localStorage`에 저장되어, 새로고침 후에도 유지됩니다.
- 총 15라운드이며 1라운드는 20마리, 이후 라운드마다 필요한 처치 수가 2마리씩 늘어납니다.
- 매 라운드 종료 후 3·2·1 카운트다운이 끝나면 다음 라운드가 시작되고, 2라운드마다 물총이 추가 발사됩니다.
- 힐팩을 먹으면 체력이 1 회복되고, 각 라운드 보스를 쓰러뜨려야 다음 라운드로 진입합니다.
- 보스는 4초 주기의 원형 탄막 스킬을 쓰며 라운드가 오를수록 총알 수가 늘고, 쿨타임은 라운드마다 0.2초씩 짧아집니다.

## 🛠️ 로컬 커스터마이징 팁

- `config` 객체에서 속도를 조절하여 난이도를 쉽게 변경할 수 있습니다.
- `drawPlayer` 함수 안의 색상 코드를 바꿔 다른 악어 색상 스킨을 만들 수 있습니다.
- 추가 사운드를 넣고 싶다면 `shootBullet`나 `spawnPopEffect`에서 `Audio` 객체를 재생하세요.
